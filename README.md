Instruções
Requisitos
  - Ter instalado .Net core versão8
  - Sql server
  - Visual Studio
Como Executar o Projeto
  - Clone o projeto https://github.com/pessanitalo/Desafio-Tecnico.git
  - Será necessario alterar a string de conexão que fica no arquivo appsettings.json
  - Rodar o comando Update-Database -Context DataContext, (Precisa selecionar o projeto Desafio-Tecnico.Data
  - Rodar o Projeto
